Bilibili (mmedia)

配置

在Hexo根目录下的_config.yml文件中，添加以下配置：

使用

在Markdown文件中，使用如下语法来插入Bilibili播放器：

参数 默认 解释
aid - aid
bvid - bvid，与 aid 同时出现时以 bvid 为准
page 1 视频分P
danmaku true 是否显示弹幕
allowfullscreen allowfullscreen 是否允许全屏播放,改为其他任意值则不允许全屏播放
sandbox allow-top-navigation allow-same-origin allow-forms allow-scripts allow-popups 安全沙盒属性
width 100% 播放器宽度
max_width 850px 播放器最大宽度
margin auto 播放器居中

Aplayer (mmedia)

安装

首先确保你的Hexo环境已经安装了hexo-tag-mmedia 插件。如果还没有，请执行以下命令来安装：

配置

在Hexo根目录下的_config.yml文件中，添加以下配置：

使用

在Markdown文件中，使用如下语法来插入Aplayer播放器：

参数 默认 解释
name - 音频名
artist - 音频作者
url - 音频链接
cover - 音频封面
lrc - 音频歌词
theme - 音频主题
type auto 音频类型，可选 ‘auto’, ‘hls’, ‘normal’
autoplay false 是否自动播放
loop ‘all’ 播放器循环播放，可选 ‘all’, ‘one’, ‘none’
order ‘list’ 播放器播放顺序，可选 ‘list’, ‘random’
volume 0.7 默认音量
tlistMaxHeight - 列表最大高度
